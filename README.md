# Como instalar?
## Debian

```
apt update  && apt upgrade

apt install git python

git clone https://github.com/oporadokrl/Kinypainel.git

cd Kinypainel

python3 main.py
```

## Arch Linux

```
pacman -Syyu git python

git clone https://github.com/oporadokrl/Kinypainel.git

cd Kinypainel

python3 main.py
```

## OpenSUSE

```
zypper install python git

git clone https://github.com/oporadokrl/Kinypainel.git

cd Kinypainel

python3 main.py
```
